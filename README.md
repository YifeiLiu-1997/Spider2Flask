# Spider2Flask
Spidered english word translation from cambridge and show it into a web site built by flask.

## How to install
```
pip install flask
pip install bs4
pip install lxml
pip install requests
```

## How to run this app
```python
open main.pyw, and will open the 127.0.0.1:5000/ in your default browser
```
